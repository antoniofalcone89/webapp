Informazioni sul modello

Quando si selezionano i file csv per mostrare la disposizione in campo della squadra, questi devono indicare i calciatori nella disposizione corrispondente al modulo, da sinistra verso destra.

Esempio: il file 1.csv deve indicare il terzino sinistro, 2.csv il difensore centrale di sinistra, 3.csv il centrale destro e così via.

Per ottenere il grafico della distanza media tra difesa e attacco sono necessari 3 file csv. E' importante seguire l'esempio seguente.

Esempio: il file 1.csv deve indicare il difensore centrale sinistro (o destro), 2.csv l'altro difensore centrale, 3.csv la punta centrale.
